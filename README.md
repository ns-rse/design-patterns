# Software Design Patterns

A repository for a blog written in [Quarto](https://www.quarto.org/) about [Software Design
Patterns](https://en.wikipedia.org/wiki/Software_design_pattern).

# Contributing

If you wish to contribute please...

1. Make an issue describing the design pattern with links and ideally examples (preferably in Python but any language is
   accepted).
2. Fork this repository.
3. Make a new directory under `posts/<pattern-name>`.
4. Copy the `template/index.qmd`  to `posts/<pattern-name>`.
5. Edit the `title`, `author`, `date` and `image` fields.
6. Write your post. [Quarto](https://www.quarto.org/) is a [Markdown](https://www.markdownguide.org/) dialect
7. Commit, push and make a pull request.

I'll try and review in a reasonable time frame.
